My dotfiles
=============================================
how to use
---------------------------------------------

	git clone git://github.com/korin/home.git korin-dotfiles
	cd korin-dotfiles
	git submodule init
	git submodule update

### how to update submodules (git 1.7.3 or never)
	git pull --recurse-submodules
	git submodule update
